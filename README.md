To run root project:

cd CDF
npm install
npm run start

To run GroupChat (Port 3003)

cd GroupChat
npm install
npm run start 

To run Chat (Port 3000)

cd Chat
npm install
cd ..
cd client
npm install
cd ..
npm run dev
